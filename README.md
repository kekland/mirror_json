A **Dart** library for parsing JSON objects using `dart:mirrors`.

**Important: This library does not work in Flutter.**

## Installation

Add [`mirror_json`](https://pub.dartlang.org/packages/mirror_json) to your dependencies.
```yaml
dependencies:
    ...
    mirror_json:
```
